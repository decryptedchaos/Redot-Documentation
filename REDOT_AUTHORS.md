# Redot Engine documentation authors

Redot Engine is developed by a community of voluntary contributors who
contribute on all areas, including writing and maintaining the documentation.

It is impossible to list them all; nevertheless, this file aims at listing
the writers who contributed significant patches to this CC BY licensed
documentation on the `Redot-Documentation` repository.

GitHub usernames are indicated in parentheses, or as sole entry when no other
name is available.
